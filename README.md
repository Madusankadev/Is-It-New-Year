<h1>Run Django Application</h1>

<p>Description: My fist Django Application. This is a simple web application to show is it new year or not when user vist the site. if it's new year shows YES otherwise it shows NO. just a basic application. </p>

<h2>How to run Project</h2>

<p>Fist clone the project from Github</p>
<pre>git clone https://github.com/Madusankadev/Is-It-New-Year.git </pre>

<p>Go into cloned project</p>
<pre>cd Is-It-New-Year</pre>

<p>Now activate the python virtual Envirement</p>
<pre>py3.12/Scripts/activate</pre>

<p>Navigate to main directory. which include project</p>
<pre>cd main</pre>

<p>Finally you can run the application now</p>
<pre>py manage.py runserver</pre>

<p>You can Access site using this URL</p>
<pre>http://127.0.0.1:8000/newyear/</pre>
